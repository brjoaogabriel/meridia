#   Meridia, comunicação responsável.

O projeto Meridia é dedicado a pequenos empreendedores de boa fé que almejam entrar em contato com seus clientes atráves de diferentes plataformas.

O programa __open source__ é responsável por realizar cadastro de pessoas com seus diferentes dados e redes sociais afim de entrar em contato via chat, email ou recado, sendo essas plataformas:
>   WhatsApp
>   Facebook
>   Instagram
>   Gmail
>   Outlook
